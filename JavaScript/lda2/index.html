Algorithmia Topic Analysis

The following is an example of topic analysis in Algorithmia. Search Twitter for any string , select the number of topics and let our algorithm analyze the topics associated with your search.

  • Demo Code
  • LDA Input
  • LDA Output
Algorithmia.client(apiKey).algo("/nlp/LDA").pipe([strings, numTopics]).then(function(topics) {
    if(topics.error) return console.error(topics.error);
    console.log(topics.result);
});
