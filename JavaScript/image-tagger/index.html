How it works

Quickly Identify Features and Automatically Tag Images

Use state-of-the-art deep learning to identify features in images. Just paste in a URL above.

The Illustration Tagger microservice, one of many cloud hosted CNN deep-learning models hosted on Algorithmia, can recognize hundreds of different features in images.

This microservice is an implementation of Saito, Masaki and Matsui, Yusuke. (2015). Illustration2Vec: A Semantic Vector Representation of Illustrations. SIGGRAPH Asia Technical Briefs. 2015.. Learn more about their work here

Let us know what you think @Algorithmia or by email.

SAMPLE INPUT

import Algorithmia

input = {"image": "_IMAGE_URL_"}
client = Algorithmia.client('_API_KEY_')
algo = client.algo('deeplearning/IllustrationTagger/0.2.3')

print algo.pipe(input)
