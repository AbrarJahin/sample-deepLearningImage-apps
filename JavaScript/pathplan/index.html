Algorithmia Path Planning

In this demo we can see 2 algorithms available in the Algorithmia API used together. The first is a maze generation algorithm that uses Recursive Backtracking Search to generate the maze. We then feed the results to a path planning algorithm that finds the shortest path through the maze using Dijkstra's Algorithm.

Algorithmia.client(apiKey).algo("/kenny/MazeGen").pipe(searchString).then(function(maze) {
    if(maze.error) return console.error(maze.error);
    var pathplanInput = [maze.result.vertices, maze.result.edges, start, goal];
    Algorithmia.client(apiKey).algo("/kenny/Dijkstra").pipe(pathplanInput).then(function(path) {
        if(path.error) return console.error(path.error);
        console.log(path.result);
    });
});
